# Implementation-SPOC

(main branch) Code accompanying the Master's thesis of Ana Vicic.

This repository contains analysis scripts, notebooks, and workflows used to evaluate AlphaFold-Multimer and SPOC-based approaches for prioritization of LINE-1 ORF2p protein interaction candidates from coIP-MS datasets.

The code was developed during an internship in the LaCava Lab at Rockefeller University.
